To determine which shell you are using? echo $SHELL

To start the BASH shell? bash

To find out the BASH directory? whereis bash
bash: /bin/bash.exe /usr/bin/bash.exe /etc/bash.bashrc /etc/bash.bash_logout /usr/share/man/man1/bash.1

To install BASH shell? chsh bash-name (Where bash-name is the location of the bash shell). For example, chsh /bin/bash.exe

Print the working directory? pwd

Change the working directory? cd

To list the content of the bash profile? cat .bash_profile.

Using vi, create the dir file, for example, vi dir. type ls, exit. chmod +x dir.

Reference
