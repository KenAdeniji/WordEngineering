Unarchive the zip file, into a new folder.
Create the Contact database and its objects by running the Contact.sql sql script.
Browse the Contact folder.

Some of the features in the Contact project, include:

Session login to support a multi-user database.
Model View Controller to facilitate separation of concerns.
Cascading style sheet.
A single database maintenance stored procedure to centralize all database modification.
A single database view to act as a front for all the database tables.
