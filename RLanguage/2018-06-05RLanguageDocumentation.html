Comments Use the hashmark to precede comments.
Recall and correction of previous commands Use the up arrow key to go to previous command, and the down arrow key to go to the next command.
Executing commands from or diverting output to a file source("filename.R") will execute the commands stored in this input file. sink("record.lis") will re-direct the console's output to this external file.
Data permanency and removing objects The command objects() or ls() is for displaying the current workspace. The rm(object name) is for removing objects. Prior to exiting, you may save your objects in the .RData file, and the commands issued in the .Rhistory file.
Vectors and assignment The command x <- c(10.4, 5.6, 3.1, 6.4, 21.7) is for creating a numeric vector.
The function call assign("x", c(10.4, 5.6, 3.1, 6.4, 21.7)) will do a similar step.
Likewise, c(10.4, 5.6, 3.1, 6.4, 21.7) -> x is the same command.
